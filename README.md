# 💳 Detecção de Fraudes em Transações de Cartão de Crédito

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como atividade prática da DIO com o objetivo de aplicar técnicas de Ciência de Dados e Machine Learning para detectar possíveis fraudes em transações de cartão de crédito.

Foi utilizado um conjunto de dados real contendo milhares de transações financeiras, no qual cada registro é classificado como transação legítima ou fraude.

---

## 🎯 Objetivos

- Explorar os dados
- Identificar possíveis problemas na base
- Balancear o conjunto de dados utilizando SMOTE
- Treinar um modelo de Machine Learning
- Avaliar o desempenho do modelo
- Explicar quais variáveis tiveram maior influência na decisão

---

## 🛠 Tecnologias

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Imbalanced-Learn (SMOTE)

---

## 📊 Etapas do Projeto

- Importação das bibliotecas
- Carregamento do dataset
- Análise Exploratória (EDA)
- Balanceamento dos dados
- Treinamento do Random Forest
- Avaliação do modelo
- Explicabilidade

---

## 📈 Resultados

O modelo apresentou:

- Accuracy superior a 99%
- Recall elevado para identificação de fraudes
- Matriz de confusão demonstrando boa capacidade de detecção
- Feature Importance para explicar as decisões do modelo
